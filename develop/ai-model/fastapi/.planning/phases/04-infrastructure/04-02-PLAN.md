    - fetch_exercise_list(user_id) -> GET {WAS_BASE_URL}/api/exercise-list/{user_id} -> list[dict] (운동 리스트)
    - fetch_meal_list(user_id) -> GET {WAS_BASE_URL}/api/meal-list/{user_id} -> list[dict] (식단 리스트)
    - On HTTP 4xx/5xx -> raises ExternalServiceError(service="WAS", message=...)
    - On timeout (10s) -> raises ExternalServiceError(service="WAS", message="timeout")

    - lifespan creates httpx.AsyncClient + WASClient on startup, closes on shutdown

  <name>Task 2: AI chat request/response schemas with db_modified_flag</name>
  <files>app/schemas/chat.py, tests/test_schemas.py</files>
  <behavior>
    - AiChatRequest(user_id, user_profile, user_instruction, user_message) validated by Pydantic
    - DbModifiedFlag is Literal["none", "exercise", "meal", "profile"] enum-like
    - AiChatResponse(status, mode, data, db_modified_flag) matches DataFormat_3_ai.md spec
    - Mode-to-flag mapping: mode 2,3 -> "exercise", mode 4,5 -> "meal", mode 6 -> "profile", else -> "none"
  </behavior>
  <action>
    1. Create app/schemas/chat.py:
       - from typing import Literal, Optional, Any
       - from pydantic import BaseModel, Field
       - Import UserProfile from app.schemas.common

       - DbModifiedFlag = Literal["none", "exercise", "meal", "profile"]

       - class AiChatRequest(BaseModel):
           user_id: str
           user_profile: UserProfile
           user_instruction: str = ""
           user_message: str

       - class PlanItem(BaseModel):
           type: str
           detail: str
           value: str

       - class PlanData(BaseModel):
           date: str
           items: list[PlanItem]

       - class DbUpdate(BaseModel):
           field: str
           new_value: Any

       - class AiChatData(BaseModel):
           message: str
           plan: Optional[PlanData] = None
           db_update: Optional[DbUpdate] = None

       - class AiChatResponse(BaseModel):
           status: str = "success"
           mode: int
           data: AiChatData
           db_modified_flag: DbModifiedFlag

       - def get_db_modified_flag(mode: int) -> DbModifiedFlag:
           """Mode-to-flag deterministic mapping. FastAPI decides, not Gemini."""
           mapping = {2: "exercise", 3: "exercise", 4: "meal", 5: "meal", 6: "profile"}
           return mapping.get(mode, "none")
