1. app/clients/pinecone.py 생성:
   - PineconeClient 클래스
   - __init__(self, index: IndexAsyncio) -> None: self._index = index 저장
   - async def upsert(self, user_id: str, vector: list[float], summary: str) -> str:
     - id = str(uuid4()) 생성
     - metadata = {"user_id": user_id, "summary": summary, "timestamp": datetime.utcnow().isoformat()}
     - await self._index.upsert(vectors=[{"id": id, "values": vector, "metadata": metadata}], namespace=user_id)
     - return id
   - async def search(self, user_id: str, vector: list[float], top_k: int = 3) -> list[dict]:
     - result = await self._index.query(vector=vector, top_k=top_k, namespace=user_id, include_metadata=True)
     - return [{"id": m.id, "score": m.score, "summary": m.metadata.get("summary",""), "timestamp": m.metadata.get("timestamp","")} for m in result.matches]

CRITICAL: namespace 파라미터를 반드시 user_id로 전달. 이것이 사용자 격리의 핵심.
벡터 ID는 UUID4 사용 (CONTEXT.md 결정).
메타데이터는 최소한: user_id, summary, timestamp만 저장 (CONTEXT.md 결정).
