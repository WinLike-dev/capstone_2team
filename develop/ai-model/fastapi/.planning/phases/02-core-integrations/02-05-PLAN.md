   - 기존 placeholder 코드를 완전히 교체

   Startup 순서 (CONTEXT.md 결정: 임베딩 -> Pinecone -> Gemini):
   a. 임베딩 모델 로드:
      - model = await run_in_threadpool(SentenceTransformer, "paraphrase-multilingual-MiniLM-L12-v2")
      - app.state.embed_client = EmbeddingClient(model)
      - logger.info("Embedding model loaded (dim=%d)", EMBEDDING_DIM)

   b. Pinecone 초기화:
      - pc = PineconeAsyncio(api_key=settings.PINECONE_API_KEY)
      - app.state._pinecone_control = pc (lifecycle 관리용)
      - if not await pc.has_index(settings.PINECONE_INDEX_NAME):
          await pc.create_index(name=settings.PINECONE_INDEX_NAME, dimension=EMBEDDING_DIM, metric="cosine", spec=ServerlessSpec(cloud="aws", region="us-east-1"))
      - description = await pc.describe_index(settings.PINECONE_INDEX_NAME)
      - index = pc.IndexAsyncio(host=description.host)
      - app.state.pinecone_client = PineconeClient(index)
      - logger.info("Pinecone initialized (index=%s)", settings.PINECONE_INDEX_NAME)

   c. Gemini 클라이언트:
      - app.state.gemini_client = GeminiClient(api_key=settings.GEMINI_API_KEY, model_name=settings.GEMINI_MODEL_NAME)
      - logger.info("Gemini client initialized (model=%s)", settings.GEMINI_MODEL_NAME)

   d. Router 클라이언트:
      - app.state.router_client = RouterClient(api_key=settings.ROUTER_API_KEY, model_name=settings.ROUTER_MODEL_NAME)
      - logger.info("Router client initialized (model=%s)", settings.ROUTER_MODEL_NAME)

   Shutdown:
   - await app.state._pinecone_control.close() (또는 __aexit__ 호출)
   - logger.info("Pinecone connection closed")

CRITICAL: 초기화 중 예외 발생 시 catch하지 않는다 — 예외를 전파하여 FastAPI가 시작을 중단하게 한다 (CONTEXT.md 결정: "초기화 실패 시: 전체 서버 시작 중단").
