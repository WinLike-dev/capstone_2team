4. app/clients/gemini.py 생성 (GEMI-01, GEMI-02):
   - GeminiClient 클래스
   - __init__(self, api_key: str, model_name: str) -> None:
     - self._client = genai.Client(api_key=api_key)
     - self._model_name = model_name
   - @retry(wait=wait_exponential_jitter(initial=1, max=60, jitter=5), stop=stop_after_attempt(5), retry=retry_if_exception_type(ResourceExhausted), reraise=True)
     async def generate(self, system_prompt: str, user_content: str, response_schema: type) -> str:
     - response = await self._client.aio.models.generate_content(model=self._model_name, contents=user_content, config=types.GenerateContentConfig(system_instruction=system_prompt, response_mime_type="application/json", response_schema=response_schema))
     - return response.text

CRITICAL: retry는 google.api_core.exceptions.ResourceExhausted만 캐치한다. 다른 에러(401, 400)는 재시도하지 않는다.
CRITICAL: response_mime_type="application/json" + response_schema를 함께 사용하여 SDK 레벨에서 JSON 출력을 보장한다.
